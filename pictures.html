Say, for the sake of example, I wanted to send the message "attack at midnight" in secret, and use the key "Basil". After letting the recipient know of the key, I would start encrypting. For the first letter, I would find the letter at the intersection of the 'A' row and 'B' column (for the phrase and key respectively), which gives me 'B'. I continue this process until all letters have been encrypted, going back to the start of the keyword whenever I go through the whole keyword. (For example, I would encrypt the message with the letters BASILB AS ILBASILB.)

The encrypted message from this example would end up being "Btlinl al utenaosu". To decrypt it knowing the key, I would start on the first letter by finding where the 'B' is on the 'B' column. In our case it is in the 'A' row, which makes sense as the first letter of "Attack at midnight" is 'A'.
